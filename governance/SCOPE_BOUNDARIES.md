## Authority Model

| Decision Type | Authority | Approver |
|---------------|-----------|----------|
| Monitoring response | Autonomous | - |
| Runbook execution | Autonomous | - |
| Non-prod deployment | Autonomous | - |
| Production deployment | Requires approval | Supervisor |
| Production changes | Requires approval | Change manager |

## Governance Intensity

**Level**: Rigorous (capability-governance-rigorous)

- Full protocols for production access
- Rollback plan required
- Change documentation mandatory
